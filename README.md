🚗 Traffic Dodger

Traffic Dodger is a fast-paced 2D car-dodging game built using Python and Pygame, where the player must skillfully navigate through traffic and avoid oncoming vehicles for as long as possible. The longer you survive, the higher your score — but beware, the speed keeps increasing!

🎮 Gameplay

Control your car using Left and Right arrow keys

Dodge randomly spawning enemy cars

Stay within road boundaries

Game difficulty increases over time with rising enemy speed

The game ends on collision or if you go off the road

⭐ Features

Smooth scrolling road animation

Increasing difficulty for an engaging challenge

Realistic sound effects (engine, acceleration, crash)

Background music support

High score system saved locally

Simple Tkinter-based login/menu screen

Clean and colorful car sprites

🛠️ Built With

Python

Pygame – game mechanics & graphics

Tkinter – game launcher UI

Pygame Mixer – sound & music handling

🧠 Learning Outcomes

This project helped reinforce concepts like:

Game loops and event handling

Collision detection

File handling for persistent high scores

Integrating sound and UI with gameplay

▶️ How to Run

Install dependencies:

pip install pygame

Run the launcher:

python Login.py

Click Play Game and start dodging traffic!

also change the paths of the images of the vehicles etc
